唤醒词自定义snowboy

什么是snowboy

snowboy是一个基于DNN深度神经网络的热词检测，常用于做语音唤醒，比如小爱音箱，小度小度等之类的唤醒设备应用场景。

snowboy支持训练自己的唤醒词，在2021年之后，snowboy关闭了在线训练自定义唤醒词，但是提供了另一个本地离线训练的模型，一起来验证一下吧。

目前支持的平台：

  • all versions of Raspberry Pi (with Raspbian based on Debian Jessie 8.0)
  • 64bit Mac OS X
  • 64bit Ubuntu 14.04
  • iOS
  • Android
  • ARM64 (aarch64, Ubuntu 16.04)

验证环境

系统：ubuntu16.04

通过https://github.com/Kitt-AI/snowboy原本的开源地址，我们找到它提供的离线训练唤醒词支持的版本：https://github.com/seasalt-ai/snowboy

环境搭建

修改python环境为国内源

pip config set global.index-url表示配置下载源，以下选择一个执行，选下载速度快的，我这里的阿里的源会快一点。

# 清华源
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
# 阿里源
pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
# 腾讯源
pip config set global.index-url http://mirrors.cloud.tencent.com/pypi/simple
# 豆瓣源
pip config set global.index-url http://pypi.douban.com/simple/

创建python的虚拟环境

sudo pip install virtualenv

初始化虚拟环境目录

virtualenv -p python2 venv/snowboy
source venv/snowboy/bin/activate

进入虚拟环境

source venv/snowboy/bin/activate

下载源代码

git clone https://github.com/seasalt-ai/snowboy.git

下载依赖项

cd snowboy-master/examples/Python
pip install -r requirements.txt

以上安装依赖pyaudio时，会提示pyaudio出现"portaudio.h":No such file or directory错误，缺少portaudio库的支持，

可以使用以下两种安装方式：

  1. 使用apt方式安装
sudo apt-get install portaudio19-dev python-all-dev python3-all-dev

  1. 使用源代码安装

    下载源代码：wget http://files.portaudio.com/archives/pa_stable_v190600_20161030.tgz

    解压源代码：tar -xzvf pa_stable_v190600_20161030.tgz

    进入源代码：cd portaudio

    编译源代码：make

    安装源代码：sudo make install

安装swig和音频库

sudo apt-get install swig3.0 sox
sudo apt-get install libatlas-base-dev

swig编译环境

进入snowboy源代码的swig/Python目录下

cd snowboy-master/swig/Python

执行

make && sudo make install

生成snowboydetect.py和_snowboydetect.so文件。

snowboy唤醒词验证

唤醒词录制

rec -r 16000 -c 1 -b 16 -e signed-integer -t wav record1.wav
rec -r 16000 -c 1 -b 16 -e signed-integer -t wav record2.wav
rec -r 16000 -c 1 -b 16 -e signed-integer -t wav record3.wav

通过以上rec命令录制三段唤醒词，比如叫”芝麻开门”

离线唤醒词训练

进入snowboy-master/examples/Python目录

python generate_pmdl.py -r1=record1.wav -r2=record2.wav -r3=record3.wav -lang=zh -n=hotword.pmdl

生成模型文件hotword.pmdl

验证唤醒词

python demo.py hotword.pmdl

启动程序后，对着麦克风喊唤醒词”芝麻开门”，成功唤醒会有”ding”的一声和一条唤醒成功的打印语句，如下：

Listening... Press Ctrl+C to exit
INFO:snowboy:Keyword 1 detected at time: 2021-04-11 20:49:35

快来自定义一个唤醒词试试吧
